# 🖼️ 图片处理工具 - 专业在线图片编辑网站

一个功能强大的在线图片处理工具网站，专注于图片编辑和处理。

## ✨ 功能特性

### 📐 基础调整
- **图片压缩** - 减小文件大小，保持质量
- **尺寸调整** - 自定义宽度和高度，支持保持宽高比
- **图片裁剪** - 精确裁剪指定区域
- **格式转换** - 支持 JPEG、PNG、WEBP、GIF 格式转换

### 🔄 旋转翻转
- **旋转** - 左旋转90°、右旋转90°
- **翻转** - 水平翻转、垂直翻转

### 🎨 滤镜效果
- **黑白** - 灰度滤镜
- **怀旧** - 复古色调
- **模糊** - 柔化效果
- **反色** - 颜色反转
- **亮度调整** - 0-200% 可调
- **对比度调整** - 0-200% 可调
- **饱和度调整** - 0-200% 可调

### 📦 批量处理
- **批量压缩** - 同时处理多张图片
- **批量调整尺寸** - 统一调整多张图片尺寸
- **批量转换格式** - 批量转换图片格式

## 🚀 快速开始

### 本地运行

1. **直接打开**
   - 在浏览器中打开 `index.html` 即可使用

2. **使用本地服务器（推荐）**
```bash
# Python 3
python3 -m http.server 8000

# Node.js (如果已安装)
npx serve .

# PHP
php -S localhost:8000
```

3. 访问 `http://localhost:8000`

## 🌐 GitHub Pages 部署

### 快速部署步骤

1. **创建 GitHub 仓库**
   - 在 GitHub 上创建一个新的公开仓库

2. **推送代码**
```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
git push -u origin main
```

3. **启用 GitHub Pages**
   - 进入仓库 Settings > Pages
   - Source 选择 "GitHub Actions"
   - 保存后会自动部署

4. **访问网站**
   - 部署完成后访问：`https://YOUR_USERNAME.github.io/YOUR_REPO_NAME/`

详细部署说明请查看 [DEPLOY.md](./DEPLOY.md)

# 🖼️ 图片处理工具 - 专业在线图片编辑网站

一个功能强大的在线图片处理工具网站，专注于图片编辑和处理。

## ✨ 功能特性

### 📐 基础调整
- **图片压缩** - 减小文件大小，保持质量
- **尺寸调整** - 自定义宽度和高度，支持保持宽高比
- **图片裁剪** - 精确裁剪指定区域
- **格式转换** - 支持 JPEG、PNG、WEBP、GIF 格式转换

### 🔄 旋转翻转
- **旋转** - 左旋转90°、右旋转90°
- **翻转** - 水平翻转、垂直翻转

### 🎨 滤镜效果
- **黑白** - 灰度滤镜
- **怀旧** - 复古色调
- **模糊** - 柔化效果
- **反色** - 颜色反转
- **亮度调整** - 0-200% 可调
- **对比度调整** - 0-200% 可调
- **饱和度调整** - 0-200% 可调

### 📦 批量处理
- **批量压缩** - 同时处理多张图片
- **批量调整尺寸** - 统一调整多张图片尺寸
- **批量转换格式** - 批量转换图片格式

## 🔧 技术栈

- **前端**: HTML5, CSS3, JavaScript (ES6+)
- **图片处理**: Canvas API
- **样式**: 纯CSS，响应式设计
- **无依赖**: 无需任何外部库（除CDN资源）
